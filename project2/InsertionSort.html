Insertion Sort

This algorithm takes a list and sorts it using CLRS's Insertion Sort algorithm. It does a linear scan to find the minimum and swaps that with the current index. N passes of N items means this algorithm has a worst case runtime of O(n^2).


for j = 2 to A.length
  key = A[j]
  //Insert A[j] into the sorted sequence A[1..j-1]
  while i > 0 and A[i] > key
    A[i+1] = A[i]
    i = i - 1
  A[i+1] = key
